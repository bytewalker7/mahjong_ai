# 麻将规则（平胡麻将版）

## 一、基本信息

### 1. 牌型

使用：

- 万子
- 条子
- 筒子

共：

```
3种花色 × 9个数字 × 4张 = 108张
```

不使用：

- 风牌
- 箭牌
- 花牌


---

# 二、游戏流程

## 1. 发牌

四名玩家参与。

每人：

```
13张牌
```

庄家：

```
14张牌
```

庄家先出牌。


---

# 三、下炮子规则

## 1. 下炮子

庄家出第一张牌之前，每个玩家可以选择是否下炮子。

规则：

- 炮子数量：0～4个
- 可以不下炮子
- 下定后本局不可修改


## 2. 炮子结算

炮子不是倍率，而是额外分数。

每个炮子：

```
1分
```

最终输赢：

```
最终分数 = 胡牌分 + 杠分 + 炮子分
```

例如：

玩家下：

```
3个炮子
```

本局结束：

- 赢：额外增加3分
- 输：额外扣除3分


---

# 四、牌型与胡牌规则

## 1. 基础胡牌

所有胡牌均称：

```
平胡
```

固定：

```
1分
```

不计算其他番型：

- 清一色
- 七对子
- 十三幺
- 豪华七对
- 大三元
- 其他特殊牌型


---

## 2. 胡牌结构

标准结构：

```
4组牌 + 1对将
```

例如：

```
123万
456万
789条
555筒
88万
```


---

# 五、碰规则

## 碰

三张相同牌：

例如：

```
555万
```

别人打出：

```
5万
```

可以碰。


---

# 六、杠规则

## 1. 暗杠

手中有四张相同牌：

例如：

```
1111万
```

直接杠。

收益：

```
+2分
```


---

## 2. 放杠

自己有三张：

```
555条
```

别人打出：

```
5条
```

形成杠。

收益：

```
+1分
```


---

## 3. 蓄杠（补杠）

已经碰：

```
555条
```

之后摸到：

```
5条
```

补杠。

收益：

```
+1分
```


---

# 七、胡牌方式

## 1. 放胡（点炮）

别人打出的牌让你胡：

收益：

```
+1分
```


点炮玩家支付。


---

## 2. 炸弹（自摸）

自己摸牌胡：

收益：

```
+2分
```

其他玩家各支付2分。


---

## 3. 截胡

多人同时可以胡同一张牌：

按照逆时针顺序：

距离出牌者最近的玩家优先胡。


---

## 4. 漏胡

如果玩家可以胡某张牌但选择放弃：

在没有重新轮到自己行牌之前：

不能再次胡同一张牌。


---

# 八、庄家规则

庄家胡牌：

胡牌分翻倍。

例如：

普通胡：

```
1分
```

庄家胡：

```
2分
```


---

# 九、最终结算

最终输赢：

```
最终分数 = 胡牌分 + 杠分 + 炮子分
```


## 胡牌分

|方式|分数|
|-|-|
|点炮|1分|
|自摸|2分|


## 杠分

|类型|分数|
|-|-|
|暗杠|2分|
|放杠|1分|
|蓄杠|1分|


## 炮子分

```
炮子数量 × 1分
